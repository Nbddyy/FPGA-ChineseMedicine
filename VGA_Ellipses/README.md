4 监控分屏

## 设计思路:bookmark_tabs:

```
二维视图中，落在椭圆中的点满足如下公式：(h,k)为椭圆的圆心坐标，a和b分别是椭圆在x和y方向上的半长轴和半短轴
(x-h)^2 / a^2 + (y-k)^2 / b^2 = 1
```
